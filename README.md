# 반짝별 탐정단

8살 아이가 그림을 보며 현장을 조사하고, 팝업으로 나타나는 단서와 대화를 따라 범인을 찾는 모바일 대응 브라우저 탐정 게임입니다.

## 실행

`index.html` 파일을 브라우저에서 열면 바로 실행됩니다. 별도 서버나 빌드 과정은 필요하지 않습니다.

GitHub Pages로 배포하면 모바일에서 홈 화면에 추가해 앱처럼 실행할 수 있습니다. 한 번 접속한 뒤에는 주요 파일이 캐시되어 오프라인에서도 다시 열 수 있습니다.

## 게임 흐름

1. 현재 장소에서 용의자 한 명을 만납니다.
2. 장소 안으로 들어간 뒤 화면의 반짝이는 영역을 직접 누릅니다.
3. 간단한 퀴즈를 맞히면 힌트 아이템을 획득합니다.
4. 힌트 아이템이 많아질수록 범인 후보가 줄어듭니다.
5. 장소 안의 인물을 직접 눌러 알리바이 대화를 듣습니다.
6. 그 사람이 범인인지 아닌지 바로 추리합니다.
7. 추리가 틀리면 해당 사건을 처음부터 다시 시작합니다.
8. 추리가 맞고 범인이 아니면 다음 장소로 이동합니다.
9. 범인을 맞히면 축하 메시지와 함께 사건이 해결됩니다.
10. 첫 사건을 해결하면 두 번째 사건인 인형극 속 가짜 사건이 열립니다.

새 게임 버튼을 누를 때마다 첫 사건부터 다시 시작하며, 각 사건의 범인은 랜덤으로 바뀝니다.
